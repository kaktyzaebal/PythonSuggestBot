Бот для создания предложки для Телеграм-канала.

НАЧАЛО РАБОТЫ С РЕПОЗИТОРИЕМ:
1. Создайте бота через BotFather и подготовьте:
  1.1. API-токен бота
   1.2. ID вашего профиля в ТГ (через любого бота для проверки ID профиля)
   1.3. ID вашего ТГ-канала (тоже самое)
2. Вставьте данные в config.py
3. Если запускаете скрипт впервые - проверьте наличие библиотек из requirements.txt
4. Запустите bot.py в любом интерпретаторе/start.sh на любом хостинге вэб-сервисов
5. Готово!
